Portfolio Website Report - CSY1063

Project Overview

This portfolio website was developed as part of the CSY1063 Web Development module. The site demonstrates fundamental web development skills including semantic HTML5, modern CSS3, responsive design, and accessibility best practices.

Technical Implementation

HTML Structure

  • Five interconnected HTML pages with consistent navigation
  • Semantic HTML5 elements (header, nav, main, section, footer)
  • Proper document structure and meta tags for SEO
  • Accessible image alt text and ARIA labels where needed

CSS Styling

  • Single external stylesheet for consistent design across all pages
  • Mobile-first responsive design using Flexbox and Grid
  • Custom CSS variables for maintainable color scheme
  • Smooth transitions and interactive hover states

Responsive Design

  • Media queries for tablet and desktop breakpoints
  • Flexible layouts that adapt to different screen sizes
  • Optimized typography scale for readability across devices

Page Structure

  • Home (index.html): Landing page with featured work and two images
  • Projects (projects.html): Interactive project showcase with clickable buttons using CSS radio inputs
  • Video Demo (skills.html): Video demonstration page with embedded media player
  • Site Report (about.html): Technical documentation and project analysis
  • Contact (contact.html): Contact page with clickable image and link to projects

Interactive Features

The Projects page implements pure HTML/CSS interactivity using hidden radio inputs and the :checked pseudo-class. This demonstrates advanced CSS techniques without requiring JavaScript, showing project details when buttons are clicked.

Design Principles

The website follows a professional, minimalist aesthetic with a monochromatic color scheme. Typography uses system fonts for optimal performance, and consistent spacing creates visual hierarchy throughout the site.
